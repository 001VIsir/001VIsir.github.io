ArrayList vs 数组 vs LinkedList

特性 数组 ArrayList LinkedList
初始化机会 固定 动态扩展 动态扩展
存储方式 连续内存 连续内存（Object[]） 链表节点（离散内存）

场景 说明

核心机制

ArrayList：基于 Object[]，扩容时创建新数组（1.5倍），复制元素

LinkedList：基于双向链表（Node prev/item/next），无容量概念

性能关键：ArrayList 的连续内存带来更好的 CPU 缓存命中率，实际通常比 LinkedList 快。
